プロジェクト

Twitter Clone

このプロジェクトは、Twitterに触発された簡易版であり、そのコンテンツと機能を再現しています。フロントエンドとバックエンドを分離したアプローチを用いて開発されています。

ユーザーはこのウェブサイトを通じて、登録、ログイン、ツイート、閲覧、返信、いいね、ユーザープロフィールの編集、フォロー/フォロー解除、そして管理者になることができます。

ユーザー名「user2」とパスワード「12345678」を入力してログインしてみてください。

See Live Source Code

Wordle Clone

このプロジェクトは、Reactを使用して、create-next-appを使ってプロジェクトを開始し、JSX構文を使い、Wordleのコンテンツと機能を再現することを目指したものです。簡易版を作成しています。

5つの英字で構成された単語を入力してください。6回のチャンスがあります。

See Live Source Code

Toast Component UI

このプロジェクトは、Reactとcreate-next-appを出発点としています。JSX構文を活用して、一般的なUIコンポーネントであるToastsを作成しています。Toastsは、簡潔で一時的な通知を表示するために設計されています。

技術ブログ

このブログでは、Next.jsとTailwind CSSが組み合わさってダイナミックでカスタマイズ可能なウェブサイトを作成する方法をご紹介します。Tailwind 3.0のプライマリーカラーアトリビュートを使用して、ブログのスタイルを簡単にカスタマイズし、好みに合わせることができます。さらに、暗いテーマと明るいテーマの両方を楽しんで、ユーザーエクスペリエンスを向上させることができます。このブログは、私の学びの記録と思考を共有するプラットフォームとして機能しています。
